# Design Taste
tags: design frontend taste ui ux slop

## Defaults

- Web apps should feel like real products, not demo landing pages.
- Prefer dense, useful interfaces for dashboards and SaaS tools.
- Avoid generic AI slop: purple gradients, glow blobs, meaningless cards, oversized empty heroes, vague icon grids.
- Use visual hierarchy, strong spacing discipline, accessible contrast, and responsive layouts.
- Use AI Council when visual quality or product direction is uncertain.
