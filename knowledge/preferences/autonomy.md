# Autonomy Preferences
tags: autonomy yolo gates

## Default

Run in `full-yolo` mode.

## Continue Without Asking

- Planning details with obvious web-app defaults.
- Parallel execution when file ownership is clear.
- Safe test/build/lint commands.
- Safe local fixes for verified bugs.
- Low-risk security hardening.

## Hard Stop

- Destructive filesystem/git/database actions.
- Production deployment without credentials or target confirmation.
- External skill/plugin installation.
- Unresolved critical security issue.
- Repeated verification failure after repair attempts.
