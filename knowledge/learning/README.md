# A-Stack Learning
tags: learning self-learning self-optimization ruvector sona reasoningbank

## Source Model

A-Stack uses a lightweight version of RuVector's self-learning loop:

- record trajectories
- score outcomes
- promote repeated successful patterns
- recommend patterns before future work
- write optimization reports

Markdown is the human-readable source. JSON files in this folder are derived operational state.

## Files

- `trajectories.md`: append-only session and task outcomes.
- `patterns.md`: promoted reusable patterns and avoid-list lessons.
- `.learning-state.json`: derived state used by scripts.
- `optimizer-report.md`: latest consolidation report.
