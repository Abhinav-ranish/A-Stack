# Find Skill Workflow

Use when no local A-Stack workflow fits, or the user asks for a specialized external skill.

## Flow

1. Check `knowledge/skills/catalog.md`.
2. Check installed local skills if available.
3. Use the installed `find-skills` skill when external discovery is needed.
4. Prefer reputable sources and inspect quality before recommending installation.

Do not install unknown skills automatically in full-yolo mode. Skill installation changes the agent environment and should be confirmed.
